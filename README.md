# Parking Lot Management System 🚗

A sleek, single-page dynamic web application designed to manage a 100-spot parking facility. Built with Vanilla JavaScript and styled with a modern aesthetic, this tool acts as a dashboard for parking attendants.

## 🌟 Key Features
* **Interactive Grid:** A live visual representation of 100 parking spots, color-coded for availability (Green = Available, Red = Occupied).
* **Live Statistics:** Real-time calculation of total spots, available spots, occupied spots, and total daily revenue.
* **Vehicle Check-In/Out:** Dynamically assign vehicles to available spots based on vehicle type (Car, Motorcycle, Truck) with variable hourly rates.
* **Search Functionality:** Instantly locate parked vehicles by license plate. The system will highlight the specific spot on the grid.
* **Responsive UI:** A beautifully styled, mobile-responsive dashboard utilizing CSS Grid and Flexbox.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Logic:** Vanilla JavaScript (ES6+ Classes and Data Structures)
* **Frameworks:** TailwindCSS (via CDN for rapid utility classes)
